# a-fitness-guide-for-college-students

> Start your workout without being  confused.

# 健身入口 1.0

> 终于能学会自己锻炼了

**作者:曹旭**

本来打算确定完全没有问题再发布的,，但转念一想： 既然这本书的目的就是为了帮助大家快速入门健身，那为什么不早点让它发挥作用呢？ 觉得对大家有用，我就先放出来了。

我一直觉得，有一个 70 分的方案尽快开始行动、反馈、改进、成长，比等一个不知道什么时候才会出现的 90 分答案出现再开始行动要好得多。
